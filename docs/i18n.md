# 🌐 Sistema de Internacionalización (i18n) Dinámico

Este proyecto utiliza un sistema de traducción automática basado en la API de Google Translate, diseñado para mantener la base de código limpia (español por defecto) y traducir el contenido en tiempo de ejecución.

## 🚀 Cómo usar

### 1. Traducir textos simples
Para que un texto sea traducible, envuélvelo con el componente `<T />`.

```tsx
import { T } from "@/components/T";

export default function Welcome() {
  return (
    <h1>
      <T>Bienvenido a Reserva Latina</T>
    </h1>
  );
}
```

### 2. Selector de Idioma
El componente `LanguageToggle` ya está disponible para ser usado en el Header o cualquier parte de la UI.

```tsx
import { LanguageToggle } from "@/components/shared/LanguageToggle";

export function Header() {
  return (
    <header>
      {/* ... otros elementos */}
      <LanguageToggle />
    </header>
  );
}
```

### 3. Traducción Manual (Hooks)
Si necesitas traducir algo programáticamente (por ejemplo, un mensaje de error o una notificación):

```tsx
import { useLanguage } from "@/lib/LanguageContext";

export function MyComponent() {
  const { t } = useLanguage();

  const handleAction = async () => {
    const translatedMsg = await t("Operación completada con éxito");
    alert(translatedMsg);
  };
}
```

## 🛠 Detalles Técnicos

- **Idiomas Soportados**: ES, EN, FR, DE, IT, PT, JA, ZH, KO.
- **Caché**: Se ha implementado un sistema de caché en memoria para evitar llamadas redundantes a la API.
- **Persistencia**: El idioma seleccionado se guarda automáticamente en el `localStorage` del navegador.
- **Arquitectura**:
  - `src/lib/LanguageContext.tsx`: Núcleo de la lógica y estado.
  - `src/components/T.tsx`: Componente envoltorio (Client Component).
  - `src/components/shared/LanguageToggle.tsx`: Interfaz de usuario para cambiar idiomas.

## ⚠️ Consideraciones
- Las traducciones son automáticas. Para términos muy específicos de la industria (gastronomía latina), se recomienda monitorizar la calidad en idiomas no latinos.
- El componente `<T>` muestra un efecto de pulsación (`animate-pulse`) suave mientras la traducción está en camino.
