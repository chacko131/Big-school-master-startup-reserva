# `public`

Carpeta de contenido y tipos para las superficies públicas.

## Responsabilidad

Centralizar datos estáticos, factories de demo y contratos de UI para las primeras páginas visibles sin autenticación.

## Qué contiene

- contenido de landing
- datos de demo para perfiles públicos de restaurante
- configuración de formularios de reserva
- pasos de confirmación y gestión pública
