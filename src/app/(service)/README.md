# (service) — Grupo de rutas del módulo de servicio independiente

Agrupa todas las rutas bajo `/service` usando el sistema de route groups de Next.js.
No comparte layout, header ni sidebar con el dashboard principal.

## Propósito
Superficie operativa exclusiva para el equipo de sala (meseros, cocina, barra, manager).
Cada rol accede a una vista distinta. El acceso se controla en el layout raíz del grupo.
