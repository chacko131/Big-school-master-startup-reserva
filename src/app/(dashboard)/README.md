# `(dashboard)`

Carpeta del panel interno del restaurante.

## Responsabilidad

Agrupar las rutas protegidas por autenticación y tenant guard para la operación diaria y la configuración inicial del tenant.

## Qué agrupa

- onboarding autenticado
- resumen operativo
- reservas
- huéspedes
- mesas
- schedule
- settings
- billing
- equipo
- notificaciones
- integraciones
- analítica
