# `catalog/domain`

Capa de dominio del módulo `catalog`.

## Responsabilidad

Definir las entidades, reglas de negocio y errores del catálogo sin depender de Prisma, Next.js ni de la infraestructura.

## Qué vivirá aquí

- entidades del restaurante
- reglas de configuración operativa
- invariantes del negocio
- errores de dominio
