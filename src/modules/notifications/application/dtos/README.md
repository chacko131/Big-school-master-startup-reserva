# `notifications/application/dtos`

Objetos de transferencia de datos para los casos de uso de notificaciones.

## Qué vivirá aquí

- `notify-member-accepted.dto.ts` — input/output del caso de uso `NotifyMemberAccepted`
- futuros DTOs para cada caso de uso de notificación

## Regla

Cada caso de uso tiene su propio DTO.
No se reutilizan DTOs entre casos de uso distintos.
