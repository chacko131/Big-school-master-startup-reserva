# Value Objects — Billing Domain

Esta carpeta contiene los objetos de valor del módulo de facturación.

- **access-level.ts** — Define los niveles de acceso del restaurante según su estado de suscripción (completo, gracia, solo lectura, suspendido) y la lógica pura que los calcula.
- **plan-features.ts** — Define qué secciones del dashboard incluye cada plan (Básico vs Pro) y cuáles son siempre accesibles independientemente del plan.
