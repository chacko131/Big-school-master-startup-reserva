# Errores de Dominio: Billing

Esta carpeta centraliza las excepciones específicas relativas al dominio de planes y suscripciones.

## Errores
- **`SubscriptionNotFoundError`**: Ocurre cuando se busca la suscripción de un restaurante y esta no está registrada.
- **`BillingDomainError`**: Excepción base de negocio para el módulo de facturación.
