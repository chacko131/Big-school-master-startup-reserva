# `public`

Carpeta de componentes reutilizables para la experiencia pública.

## Responsabilidad

Contener bloques visuales escalables para landing, perfil público, formulario de reserva y pantallas post-reserva.

## Principios

- componentes pequeños y reutilizables
- datos tipados desde `src/lib/public`
- compatibilidad con el sistema de traducción automática
- sin lógica de negocio dentro de la UI
