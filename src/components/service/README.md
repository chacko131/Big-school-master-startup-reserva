# service — Componentes del módulo de servicio independiente

Componentes de presentación para el módulo `/service`.
Completamente independientes del dashboard — sin imports de `@/components/dashboard`.

## Subdirectorios

- **`layout/`** → Header inteligente del módulo: manager (con switcher Sala/Cocina/Barra) y staff (solo su área).
- **`kds/`** → Componentes del Kitchen Display System (modo oscuro).
- **`floor/`** → Componentes de la vista de sala del mesero.
- **`overview/`** → Componentes de la vista de control total del dueño/manager.
