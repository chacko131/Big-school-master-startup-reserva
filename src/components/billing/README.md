# Billing Components

Esta carpeta contiene los componentes de interfaz de usuario (UI) reutilizables para la facturación, los periodos de prueba y los planes de suscripción que visualizan los restaurantes tenants desde su dashboard.

## Componentes principales:
- `PlanCard`: Muestra las características y el precio mensual de un plan de suscripción específico.
- `CurrentPlanBanner`: Informa sobre la suscripción activa, días restantes de trial o vigencia y estado actual del tenant.
- `PlanSelector`: Compone y organiza las tarjetas de planes y el banner del plan actual.
