# Componentes de Analíticas (`src/components/analytics`)

Esta carpeta contiene los componentes de presentación y UI utilizados para estructurar el panel de analíticas operativas y de comportamiento de clientes del restaurante.

## Componentes Disponibles

1.  **`AnalyticsToolbar.tsx`**: Renderiza la cabecera del panel de analíticas, permitiendo navegación y contextualización del servicio.
2.  **`AnalyticsMetricsGrid.tsx`**: Muestra las tarjetas con KPIs reales del restaurante (reservas totales, comensales sentados, tasa de no-shows y tasa de cancelaciones).
3.  **`AnalyticsZonesPanel.tsx`**: Representa la distribución real del flujo de reservas y comensales en las distintas zonas operativas del salón, con barras de progreso dinámicas.
4.  **`AnalyticsInsightRail.tsx`**: Genera de forma automatizada y dinámica recomendaciones e insights operativos de negocio en base al comportamiento real de las reservas y huéspedes en el periodo consultado.
