# Componentes de Reserva (Flow)

Esta carpeta contiene los componentes utilizados durante el proceso de reserva paso a paso (Wizard).

## Estructura

- `ReservationHero.tsx`: Encabezado del flujo de reserva con el nombre del restaurante.
- `PartySizePicker.tsx`: Selector de número de personas.
- `DatePicker.tsx`: Calendario de selección de fecha.
- `TimeSlotPicker.tsx`: Selector de horas disponibles.
- `ContactForm.tsx`: Formulario para capturar los datos del cliente.
- `ReservationSummary.tsx`: Resumen lateral de la reserva en proceso.
