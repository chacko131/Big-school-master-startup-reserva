# Componentes de Confirmación (Guest)

Esta carpeta contiene los componentes específicos para la página de confirmación de reserva del flujo de invitados.

## Estructura

- `ConfirmationHero.tsx`: Cabecera con mensaje de éxito y nombre del restaurante.
- `ConfirmationSummaryCard.tsx`: Tarjeta con los detalles clave de la reserva (Fecha, Hora, Personas, ID).
- `ConfirmationLocationCard.tsx`: Información de ubicación con mapa estático.
- `ConfirmationActionButtons.tsx`: Botones de acción rápida (Calendario, Gestionar, Compartir).
- `ConfirmationPromoCard.tsx`: Sección de upsell para experiencias personalizadas o menús especiales.
