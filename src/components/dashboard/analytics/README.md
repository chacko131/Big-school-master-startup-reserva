# analytics — Componentes del panel de analíticas del dashboard

Componentes de presentación para las distintas secciones de analítica del dashboard.
Todos son componentes puros — reciben datos del Server Component (page.tsx) y los renderizan.

## Subdirectorios

- **`service/`** → Componentes del panel de analíticas de servicio (KPIs, gráfico de barras, tabla top 10, heatmap, insights).
